## `details` 下 html 代码

* **粗体**

  ```html
  <b>粗体</b>
  ```

* *斜体*

  ```html
  <i>斜体</i>
  ```

* <u>下划线</u>

  ```html
  <u>下划线</u>
  ```

* ~~删除线~~

  ```html
  <s>删除线</s>
  ```

* 图片

  ```html
  <img src="">
  ```

* 分割线

  ```html
  <hr>
  ```

* 标题

  `<h1>` 至 `<h6>`，但是不建议。

  建议使用 `style="font-size:30px;font-weight:bolder"` 来代替。

  否则会影响目录。
